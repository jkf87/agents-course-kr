## Hugging Face Discord의 Agents 코스

처음 Discord를 사용하면 조금 혼란스러울 수 있습니다. 다음 안내를 참고해 편하게 사용하세요.

HF 커뮤니티 서버에는 다양한 주제의 채널이 있어 논문 토론, 이벤트 등 여러 활동에 참여할 수 있습니다.

[가입](http://hf.co/join/discord)을 마치면 `#introduce-yourself` 채널에 인사해 주세요.

Agents Course를 위해 4개의 채널을 만들었습니다:

- `agents-course-announcements`: **최신 코스 소식**을 확인합니다.
- `🎓-agents-course-general`: **일반 대화와 토론**을 나눕니다.
- `agents-course-questions`: **질문하고 서로 돕는** 채널입니다.
- `agents-course-showcase`: **멋진 에이전트를 공유**합니다.

또한 다음 채널도 참고하세요:

- `smolagents`: **라이브러리 토론과 지원**을 위한 채널입니다.

## Discord를 효과적으로 사용하려면

### 서버에 참여하는 방법

익숙하지 않다면 <a href="https://support.discord.com/hc/en-us/articles/360034842871-How-do-I-join-a-Server#h_01FSJF9GT2QJMS2PRAW36WNBS8" target="_blank">이 가이드</a>에서 서버 참여 방법을 확인할 수 있습니다.

간단히 요약하면 다음과 같습니다:

1. <a href="https://discord.gg/UrrTSsSyjb" target="_blank">초대 링크</a>를 클릭합니다.
2. Discord 계정으로 로그인하거나 새 계정을 만듭니다.
3. "I'm not a robot" 확인을 합니다.
4. 닉네임과 아바타를 설정합니다.
5. "Join Server"를 클릭합니다.

### Discord 활용 팁

- **음성 채널**도 있지만 주로 텍스트 채팅을 사용합니다.
- 코드 작성 시 유용한 **마크다운 형식**을 지원합니다. 링크에는 다소 제한적일 수 있습니다.
- 긴 대화는 **스레드**를 열어 진행하면 정리가 쉽습니다.
