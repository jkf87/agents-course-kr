### 5단계: (선택) Ollama로 로컬에서 모델 실행하기

1. **Ollama 설치**
   <a href="https://ollama.com/download" target="_blank">공식 설치 안내</a>를 참고하세요.
2. **모델 다운로드**
```bash
ollama pull qwen2:7b # 다른 모델은 Ollama 사이트 참조
```
3. **백그라운드에서 Ollama 실행**
```bash
ollama serve
```
포트 사용 오류가 발생하면 `sudo lsof -i :11434` 명령으로 사용 중인 프로세스를 확인하세요.
4. **`InferenceClientModel` 대신 `LiteLLMModel` 사용하기**
```bash
pip install 'smolagents[litellm]'
```
```python
from smolagents import LiteLLMModel

model = LiteLLMModel(
    model_id="ollama_chat/qwen2:7b",
    api_base="http://127.0.0.1:11434",
    num_ctx=8192,
)
```
5. **왜 이렇게 하나요?**
- Ollama는 OpenAI 호환 API를 통해 로컬에서 모델을 제공합니다.
- `LiteLLMModel`은 OpenAI 호환 형식을 지원하는 모든 모델과 통신할 수 있습니다.
- 따라서 `InferenceClientModel`을 `LiteLLMModel`로 바꾸면 별다른 코드 수정 없이 로컬 모델을 사용할 수 있습니다.
