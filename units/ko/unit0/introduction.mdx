오늘날 가장 흥미로운 AI 주제인 **에이전트**의 세계에 오신 것을 환영합니다!

이 무료 코스에서는 AI 에이전트를 이해하고 사용하며 직접 구축하는 법을 **초보자부터 전문가까지** 단계적으로 배웁니다.

첫 번째 유닛에서는 다음 내용을 다룹니다:

- **코스 커리큘럼** 확인
- 수강 방식 선택(자율 수강 또는 인증 과정)
- 인증 절차와 마감일 안내
- 강사진 소개
- **Hugging Face 계정 생성**
- **Discord 서버 가입** 후 동료와 인사하기

그럼 시작해 볼까요?

## 이 코스에서 얻을 수 있는 것들 [[expect]]

이 코스에서는 다음을 배울 수 있습니다:

- 📖 **이론, 설계, 실습**을 통해 AI 에이전트 학습
- 🧑‍💻 [smolagents](https://huggingface.co/docs/smolagents/en/index), [LlamaIndex](https://www.llamaindex.ai/), [LangGraph](https://langchain-ai.github.io/langgraph/) 등 **주요 라이브러리 사용법** 익히기
- 💾 Hugging Face Hub에 **여러분의 에이전트 공유** 및 커뮤니티에서 만든 에이전트 탐색
- 🏆 다양한 챌린지에 참여해 **다른 학습자들과 에이전트 성능을 비교**
- 🎓 과제 완료 후 **수료증 획득**

이 외에도 많은 혜택이 있습니다!

이 코스를 마치면 **에이전트가 어떻게 동작하는지** 이해하고, 최신 라이브러리와 도구를 활용해 **직접 에이전트를 구축**할 수 있게 됩니다.

**<a href="https://bit.ly/hf-learn-agents">코스 등록</a>**을 잊지 마세요!

(개인정보는 철저히 보호됩니다. 각 유닛이 공개될 때 링크와 과제 안내를 이메일로 보내드리기 위해 주소를 수집합니다.)

## 코스 구성은 어떻게 되나요? [[course-look-like]]

코스는 다음과 같이 구성됩니다:

- *기본 이론 유닛*: 에이전트 **이론을 학습**합니다.
- *핸즈온*: **준비된 환경의 Hugging Face Spaces**에서 주요 라이브러리를 사용해 실습합니다.
- *사용 사례 과제*: 배운 내용을 실제 문제 해결에 적용합니다.
- *최종 챌린지*: 다른 학습자들과 경쟁하며 [리더보드](https://huggingface.co/spaces/agents-course/Students_leaderboard)에서 결과를 비교합니다.

이 **코스는 여러분의 피드백과 기여로 성장하는 프로젝트**입니다. [GitHub에 이슈나 PR](https://github.com/huggingface/agents-course)을 열어 의견을 남기고, Discord에서도 활발히 소통해주세요.

수강 후에는 [👉 이 설문](https://docs.google.com/forms/d/e/1FAIpQLSe9VaONn0eglax0uTwi29rIn4tM7H2sYmmybmG5jJNlE5v0xA/viewform?usp=dialog)을 통해 의견을 보내주실 수 있습니다.

## 커리큘럼은 어떻게 되나요? [[syllabus]]

아래는 **코스의 일반적인 커리큘럼**입니다. 세부 내용은 각 유닛에서 안내됩니다.

| Chapter | Topic | Description |
| :---- | :---- | :---- |
| 0 | Onboarding | 코스에 필요한 도구와 플랫폼 준비 |
| 1 | Agent Fundamentals | Tools, Thoughts, Actions, Observations 개념과 형식, LLM, 메시지, 특수 토큰, 챗 템플릿 소개. 파이썬 함수를 활용한 간단한 예시 |
| 1.5 | Bonus: Fine-tuning an LLM for function calling | LoRa를 사용해 노트북에서 함수 호출을 수행하도록 모델을 파인튜닝 |
| 2 | Frameworks | smolagents, LangGraph, LlamaIndex 등 인기 라이브러리에서 기본 개념이 어떻게 구현되는지 살펴보기 |
| 2.5 | Bonus: Agent Observability and Evaluation | 에이전트를 추적하고 평가하여 프로덕션 준비하기 |
| 3 | Use Cases | 실제 사례 구축 (경험 많은 빌더들의 PR 환영 🤗) |
| 4 | Final Assignment | 선택한 벤치마크용 에이전트 구축 및 학생 리더보드에서 역량 증명 🚀 |

주요 커리큘럼 외에도 3개의 보너스 유닛이 있습니다:
- *Bonus Unit 1*: 함수 호출을 위한 LLM 파인튜닝
- *Bonus Unit 2*: 에이전트 가시성 및 평가
- *Bonus Unit 3*: 포켓몬 게임에서의 에이전트

예를 들어 Bonus Unit 3에서는 포켓몬 배틀 에이전트를 만드는 방법을 배웁니다 🥊.

## 인증 과정 [[certification-process]]

<img src="https://huggingface.co/datasets/agents-course/course-images/resolve/main/en/unit0/three-paths.jpg" alt="Two paths" width="100%"/>

이 코스는 *자율 수강*으로 진행하거나, 과제를 수행해 *두 종류의 인증서 중 하나*를 받을 수 있습니다.

자율 수강의 경우 모든 챌린지와 과제에 자유롭게 참여할 수 있으며 **별도의 신고는 필요 없습니다**.

인증 과정은 **완전 무료**입니다:

- *기본 과정 인증서*: 유닛 1을 완료하면 받을 수 있습니다. 최신 에이전트 트렌드를 빠르게 익히고 싶은 분들께 적합합니다.
- *수료증*: 유닛 1, 추후 제시될 사용 사례 과제 중 하나, 그리고 최종 챌린지를 모두 완료해야 합니다.

모든 과제는 **2025년 7월 1일** 이전에 제출해야 합니다.

<img src="https://huggingface.co/datasets/agents-course/course-images/resolve/main/en/unit0/deadline.jpg" alt="Deadline" width="100%"/>
