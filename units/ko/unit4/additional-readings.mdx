# 이제 무엇을 배워야 할까요?

에이전틱 AI 분야는 빠르게 발전하고 있으며, 지능적이고 자율적인 시스템을 구축하려면 기본 프로토콜을 이해하는 것이 필수입니다.

특히 알아 두어야 할 두 가지 표준이 있습니다:

- **Model Context Protocol (MCP)**
- **Agent-to-Agent Protocol (A2A)**

## 🔌 Model Context Protocol (MCP)

**Model Context Protocol (MCP)**은 Anthropic에서 제안한 오픈 스탠더드로, AI 모델이 외부 도구나 데이터, 애플리케이션과 안전하고 원활하게 **연결**될 수 있게 해 더 강력하고 자율적인 에이전트를 만들 수 있게 해 줍니다.

MCP는 USB‑C 포트와 같은 **범용 어댑터**에 비유할 수 있습니다. 각 환경마다 별도의 통합 작업 없이 다양한 디지털 환경에 AI 모델을 연결할 수 있죠.

OpenAI와 구글 같은 주요 기업들도 MCP를 채택하기 시작하면서 업계 전반에서 빠르게 주목받고 있습니다.

## 🤝 Agent-to-Agent (A2A) Protocol

구글에서는 Anthropic의 MCP와 상호 보완되는 **Agent-to-Agent (A2A) 프로토콜**을 개발했습니다.

MCP가 에이전트를 외부 도구와 연결해 준다면, **A2A는 에이전트끼리 연결**해 복잡한 문제를 함께 해결하는 협력적 다중 에이전트 시스템을 구현할 수 있게 합니다.
