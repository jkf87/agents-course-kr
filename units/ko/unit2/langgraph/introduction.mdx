# `LangGraph` 소개

<img src="https://huggingface.co/datasets/agents-course/course-images/resolve/main/en/unit2/LangGraph/LangGraph.png" alt="Unit 2.3 Thumbnail"/>

이번 섹션에서는 복잡한 LLM 워크플로를 구성하고 오케스트레이션할 수 있게 도와주는 [`LangGraph`](https://github.com/langchain-ai/langgraph) 프레임워크를 사용해 **애플리케이션을 구축하는 방법**을 배웁니다.

`LangGraph`는 에이전트의 흐름을 **세밀하게 제어**할 수 있도록 해 주어 **프로덕션 준비가 된** 애플리케이션을 만들 수 있게 합니다.

</Tip>
