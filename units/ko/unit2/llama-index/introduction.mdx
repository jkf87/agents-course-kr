# LlamaIndex 소개

이 모듈에서는 [LlamaIndex](https://www.llamaindex.ai/) 툴킷을 사용해 LLM 기반 에이전트를 구축하는 방법을 배웁니다.

LlamaIndex는 **인덱스와 워크플로를 활용해 데이터 위에서 LLM 기반 에이전트를 만들 수 있는 종합 툴킷**입니다. 이 코스에서는 에이전트 구축에 도움이 되는 세 가지 주요 부분, 즉 **컴포넌트**, **에이전트와 도구**, **워크플로**에 초점을 맞춥니다.

![LlamaIndex](https://huggingface.co/datasets/agents-course/course-images/resolve/main/en/unit2/llama-index/thumbnail.png)

다음은 LlamaIndex의 핵심 요소와 이들이 에이전트에 어떻게 도움이 되는지에 대한 간략한 설명입니다:

- **컴포넌트**: LlamaIndex에서 사용하는 기본 빌딩 블록으로 프롬프트, 모델, 데이터베이스 등이 포함됩니다. 컴포넌트는 종종 LlamaIndex를 다른 도구나 라이브러리와 연결하는 역할을 합니다.
- **도구**: 검색, 계산, 외부 서비스 접근과 같은 특정 기능을 제공하는 컴포넌트입니다. 에이전트가 작업을 수행할 수 있게 하는 기본 요소입니다.
- **에이전트**: 도구를 사용해 의사 결정을 내리는 자율적인 컴포넌트입니다. 여러 도구 사용을 조율해 복잡한 목표를 달성합니다.
- **워크플로**: 단계별로 로직을 처리하는 과정입니다. 에이전틱 워크플로는 명시적으로 에이전트를 사용하지 않고도 에이전트의 행동을 구조화하는 방법입니다.

## LlamaIndex의 특별한 점은?

LlamaIndex는 smolagents 등 다른 프레임워크와 비슷한 점도 있지만, 다음과 같은 장점을 제공합니다:

- **명확한 워크플로 시스템**: 이벤트 기반 비동기 우선 문법을 사용해 에이전트의 의사 결정 과정을 단계별로 구성할 수 있습니다. 이를 통해 로직을 명확하게 조합하고 정리할 수 있습니다.
- **LlamaParse를 통한 고급 문서 파싱**: LlamaParse는 LlamaIndex용으로 제작되어 원활하게 통합되지만 유료 기능입니다.
- **다양한 준비된 컴포넌트**: LlamaIndex는 오래전부터 개발되어 많은 다른 프레임워크와 호환됩니다. 따라서 검증된 LLM, 리트리버, 인덱스 등 신뢰성 높은 컴포넌트가 풍부합니다.
- **LlamaHub**: 수백 가지 컴포넌트, 에이전트, 도구를 등록해 놓은 레지스트리로, LlamaIndex 내에서 활용할 수 있습니다.
